# Exception

Em Java, exceções são eventos que ocorrem durante a execução de um programa e interrompem o fluxo normal de execução. Esses eventos podem ocorrer por diversas razões, como erros de lógica, operações impossíveis, condições inesperadas, entre outros. As exceções em Java são representadas por classes e são parte integrante da hierarquia de classes de exceção.

A hierarquia de exceções em Java é liderada pela classe java.lang.Throwable, que possui duas subclasses principais: java.lang.Error (indicando erros graves que geralmente não são recuperáveis) e java.lang.Exception (indicando exceções que podem ser tratadas e recuperadas).

A classe Exception tem várias subclasses, como RuntimeException (indicando exceções que ocorrem durante a execução do programa) e várias outras exceções específicas para situações particulares.
