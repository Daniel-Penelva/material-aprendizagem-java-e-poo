# Polimorfismo

O polimorfismo é um conceito fundamental na programação orientada a objetos (OOP). Ele se refere à capacidade de objetos de diferentes classes serem tratados como objetos de uma mesma classe base por meio de uma interface comum. O polimorfismo permite que um objeto possa ser usado de maneira geral, sem precisar conhecer a classe específica do objeto.

O polimorfismo é uma característica poderosa da OOP, pois permite escrever código mais flexível e reutilizável, além de facilitar a manutenção e extensão do sistema.
