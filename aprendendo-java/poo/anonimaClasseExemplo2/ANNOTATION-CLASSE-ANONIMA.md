# Classe Anônima

Uma classe anônima é uma maneira de criar uma classe sem fornecer um nome explicitamente. Elas são frequentemente usadas para criar instâncias de interfaces ou classes abstratas de maneira concisa. As classes anônimas são definidas e instanciadas ao mesmo tempo, diretamente no código onde são necessárias.

As classes anônimas são úteis quando precisa de uma implementação única de uma interface ou classe abstrata em um local específico do código, evitando a criação de uma classe separada. Elas são bastante utilizadas em situações onde a implementação é pequena e localizada.
